# Captain Of Industry: Rate Calculator Mod

Calculate maximum production and consumption rates for the selected machines.

## Installation

Download the latest release from the [releases page](https://github.com/CaptainOfIndustry/RateCalculator/releases) under the Assets section.

Place the file in the `mods` folder of your game.

> The mods folder is located in `...`

## Usage

**Press y to start selecting buildings**

**F9 to (re)open the RateCalculator window**
